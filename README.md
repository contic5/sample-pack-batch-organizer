# Sample Pack Batch Organizer
## What this project did
The Sample Pack Batch organizer helps prepare Sample Pack Batches for printing.

Before a sample pack batch is printed out, an employee needs to set the batch name, remove all sample packs with invalid SKUs, remove all unimportant columns and sort by company name.

This program completes all of these steps automatically, which saves time. Sample Pack printing is done at the start of the day, so printing quickly is essential.

## Statistics
Over 4 years, the Sample Pack Batch Organizer organized 181 batches and 12968 Sample Packs.

This has saved a significant amount of time and energy.
